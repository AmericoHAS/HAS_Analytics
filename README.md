# HAS Analytics

Este repositório contém o desenvolvimento do site pessoal profissional da **HAS Analytics**, marca vinculada ao trabalho do **Prof. Me. Haward Antunny S. Américo**.

O site tem como objetivo divulgar projetos acadêmicos, trabalhos estatísticos, serviços de análise de dados, Bioestatística Aplicada, modelagem estatística, relatórios em R Markdown e consultorias voltadas a pesquisas científicas, artigos, dissertações e teses.

A proposta é construir uma página moderna, responsiva e profissional, utilizando HTML, CSS e JavaScript, com futura integração do formulário de solicitação de orçamento ao Google Sheets por meio do Google Apps Script.
